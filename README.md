# Professional Multi-Page Website

# Description
A semantic, accessible, and responsive multi-page website built using HTML5 and CSS3.

# Pages
- Home (index.html)
- About
- Projects
- Contact (form page)

# Features
- Semantic landmarks (header, nav, main, footer)
- Exactly one h1 per page
- Accessible form with labels, fieldset, legend
- Responsive images
- Embedded YouTube video
- Mobile-friendly layout
